== README

This project made for me to learn Rails and React.  

It is a simple audio player.  

You can try free from the following URL.  

https://koalab-lemonade.herokuapp.com/

There are the following conditions.

* Google Chrome or FireFox (not working on Safari)
* .m4a file only.
* 10 megabytes or less
* Uploaded audio files will be deleted after one day.
